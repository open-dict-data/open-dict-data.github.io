IPA Lookup - Search for Pronuncations in Different Languages

Finding pronunciations in IPA, even for large and otherwise well-supported languages, can be difficult and at times inconsistent. The tools below allow you to look up words and phrases in different languages and get transcription in IPA (and sometimes audio!).

The data comes from the ipa-dict project. If you would like to add a language that you don’t see here, or make a correction, check out the project page on GitHub!

Languages

  • Arabic
  • Cantonese
  • English (US)
  • Esperanto
  • Finnish
  • French
  • Jamaican Creole
  • Japanese
  • Malay (Bahasa Indonesia and Bahasa Malaysia)
  • Mandarin:
    • Simplified
    • Traditional
  • Norwegian (Bokmål - includes audio from the norsk uttale project)
  • Persian
  • Spanish:
    • Mexico
    • Spain
  • Swahili
  • Swedish

Development

Check out the IPA Lookup project on GitHub for more information or to report a bug wth the website.

If you find errors in the IPA data itself please open an issue with the ipa-dict project.
